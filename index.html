terminal Comandos /veonjs

  • /veon reloadRecarrega todos os módulos instantaneamente.

Instalação

1 Baixe o plugin VeonJS.jar v4.0.

2 Certifique-se de usar Java 21+.

3 Coloque na pasta /plugins e inicie o servidor.

4 Crie seus arquivos .java na pasta /veonjs/modules.
